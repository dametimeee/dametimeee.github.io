## 풀이

DP를 사용하면 시간초과다.<br>
이분탐색을 사용해야 한다.

```python
import sys
input = sys.stdin.readline

N = int(input())
A = list(map(int, input().split()))
items = [0]

for i in A:
    if items[-1] < i:
        items.append(i)
    else:
        left = 0
        right = len(items)
        while left < right:
            mid = (left + right) // 2
            if items[mid] < i:
                left = mid + 1
            else:
                right = mid
        items[right] = i

print(len(items) - 1)
```
