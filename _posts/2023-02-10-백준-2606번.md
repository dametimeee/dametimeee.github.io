## 풀이

DFS를 이용한다.

```python
computer = int(input())
network = int(input())

graph = [[] for _ in range(computer + 1)]
visited = [0 for _ in range(computer+1)]

for i in range(network):
    a,b = map(int,input().split())
    graph[a] += [b]
    graph[b] += [a]

def dfs(x):
    visited[x] = 1
    for i in graph[x]:
        if visited[i] == 0:
            dfs(i)

dfs(1)
print(sum(visited)-1)
```
