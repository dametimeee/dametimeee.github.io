## 풀이

```python
import sys
input = sys.stdin.readline


N = int(input().rstrip())
a = list(input().rstrip())
len_a = len(a)

for i in range(N-1):
    b = list(input().rstrip())
    for j in range(len_a):
        if a[j] != b[j]:
            a[j] = "?"

print("".join(a))
```
